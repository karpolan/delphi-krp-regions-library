   Our skin format was designed by KARPOLAN after analyzing tons of other skin formats. We hope this format will be useful and popular. Remember our skin format is open and free! Any new ideas and propositions are welcome! Send you propositions to karpolan@ABFsoftware.com.

   We decide to use 4 run-time states for all active areas of program user interface:
1. Normal - default state when area is not disabled.
2. Selected - state when cursor run ower area and area is not disabled (Hot track).
3. Active - state when mouse clicked and area responds this click as activate event (Pressed).
4. Disabled - state when area doesn't respond any mouse or other events.

   The skin should conatins images for all run-time states used in your program. Active areas of program painted depending on its run-time state, so when you run mouse over some area it will be viewed as Selected, press some area and it will be viewed as Active and so on...    We've designed a skin description file format too. This format can be changed in future but we will support all versions of it. Skin description file is a standars windows *.ini file with following section and variables:

[Pictures] - this section contains names of all skin image files and mask file. File names can be relative (Mask = ..\A\Mask.gif).
Mask = file name of mask graphic.
Normal = file name of normal image.
Selected = file name of selected image.
Active = file name of active image.
Disabled = file name of disabled image.

[FileInfo] - this section contains inforamtion about skin file.
Project = specify the project skin was designed for... We advice to use application exe name (Lens.exe for example) for that.
Description = description of this skin (Name, signature e.t.c) .
Version = version of skin.
Author = author information (Name, nickname e.t.c).
Copyright = legal copyright inforamtion.

This is an example of skin description file:

==========================================================
[FileInfo]
Project=Lens.exe
Description=The default skin for Lens.exe 
Version=1.60
Author=KARPOLAN
Copyright=Copyright © 2000 by KARPOLAN.

[Pictures]
Mask=Mask.gif
Normal=Normal.jpg
Selected=Selected.jpg
Active=Active.jpg
==========================================================
